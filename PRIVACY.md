Privacy Policy for Tred - Trivia & FunFacts Tracker

Tred collects and stores only the minimum information required to function:

**Information Collected:**
- Discord User IDs for mentions and command responses
- Discord usernames (used as contributor names)
- Trivia/funfact text submitted by users
- Category information for trivia organization
- Submission dates (in DD/MM/YYYY format)

**How Information is Used:**
- User IDs and usernames are used to attribute trivia contributions
- Trivia text is displayed to all server members via commands
- Categories are used for organization and filtering
- Dates are used for tracking and daily trivia rotation

**Data Storage:**
- All trivia data is stored locally in a trivia.json file
- No data is transmitted to external servers
- No personal information beyond Discord usernames is stored
- Data persists only in the bot's local storage

**Data Sharing:**
- No personal information is shared with third parties
- Trivia content is visible to all members of Discord servers where the bot is present
- Contributor usernames are displayed alongside submitted trivia

**User Rights:**
- Users can view all their contributed trivia using `/mine`
- Users can edit their own trivia using `/edit`
- Users can delete their own trivia using `/delete`
- The bot owner has full administrative rights over all data

**Data Retention:**
- Trivia data is retained indefinitely unless manually deleted
- Users can request deletion of their contributions at any time
- The bot owner can purge all data if needed

**Security:**
- Access to the trivia database is limited to the bot and its owner
- The bot runs on secure, private infrastructure
- Regular backups are recommended but not guaranteed

By using Tred, you consent to this Privacy Policy and understand how your data is collected, used, and stored.
